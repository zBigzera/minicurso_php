# Mini Curso de PHP — Do Zero ao CRUD

Bem-vindo ao **Mini Curso de PHP**, um guia prático para aprender a programar aplicações web dinâmicas usando PHP, do básico até a construção de um CRUD funcional com sessão e banco de dados.
Para ver exemplos de código, arrays, funções e CRUD, acesse: [Exemplos de Código](guia_basico.md)

---

## 🎯 Objetivo

Capacitar alunos a:

- Entender a linguagem PHP do zero  
- Manipular variáveis, arrays e funções  
- Trabalhar com sessões para armazenar dados temporários  
- Criar formulários e processar informações do usuário  
- Construir um CRUD simples (Create, Read, Update, Delete)  

---

## 🗂️ Conteúdo do Curso

### 1️⃣ Introdução ao PHP
- O que é PHP e como funciona no servidor  
- Diferença entre código do lado do servidor e HTML  

### 2️⃣ Ambiente de Desenvolvimento
- Instalação do PHP e servidor local (XAMPP, Laragon ou PHP embutido)  
- Estrutura básica de arquivos `.php`  

### 3️⃣ Sintaxe Básica
- Variáveis e tipos de dados  
- Operadores e expressões  
- Condicionais
- Loops
- Funções
- Arrays

### 4️⃣ PHP + HTML
- Misturando PHP e HTML  
- Formulários com `GET` e `POST`  
- Exibindo dados dinamicamente  

### 5️⃣ Sessões
- Conceito de sessão em PHP  
- Armazenando dados temporários do usuário  
- Aplicação prática: lista de compras

### 6️⃣ CRUD Básico
- Conectar PHP a banco de dados (MySQL)  
- Operações: Create, Read, Update, Delete

## 💻 Projeto Final

Ao final do curso, os alunos serão capazes de criar um **sistema CRUD**, com:
- Cadastro, alterações, exclusões via formulário  
- Armazenamento em sessão e banco de dados
- Exibição em lista
